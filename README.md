# MODELAMIENTO
Modelamiento de Problemas ADAII Segundo Proyecto

## Como ejecutar el programa
Tener minizinc.exe agregado en el PATH

Instalar tkinter para visualizacion de interfaz
```
pip install tkinter
```

Inicializar la aplicacion desde el simbolo de sistema, estando ubicado en la misma carpeta descargada
```
python PUICA.py
```
